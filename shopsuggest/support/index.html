主な使い方

買い物リストとして使う

店舗ごとに項目を分けて入力し、買い回り順や優先順位に合わせてドラッグで並べ替えできます。

家のチェックリストとして使う

買い物以外にも、家事や日常の確認リストとして利用できます。進捗表示で今の状態も把握しやすくなります。

サジェストを再利用する

過去に入力した内容はサジェストとして再利用できます。よく使う項目をテンプレートのように素早く追加できます。

見た目を調整する

文字、色、アイコン、取り消し線などを変更して、自分にとって見やすい表示に調整できます。

Pro について

無料で使える範囲

入力、変更、複数店舗対応、ドラッグ並べ替え、サジェスト再利用、進捗管理、見た目のカスタマイズは無料で使えます。

Pro で追加される機能

Pro では広告の非表示と、JSON の書き出し / 読み込みが使えるようになります。

JSON 書き出し / 読み込み

できること

  • バックアップとして保存する
  • 別の端末へ移行する
  • 保存した内容をあとで読み込む

JSON 機能は Pro 向け機能です。重要なデータは、必要に応じて複数の場所に保管することをおすすめします。
