This page explains how information is handled in ShopSuggest.

1. 取得する情報

ShopSuggest では、ユーザーが入力した買い物リスト、チェックリスト、表示設定、保存データなどを、 アプリの機能提供のために取り扱います。

  • 買い物リストやチェックリストの入力内容
  • 店舗名、商品名、進捗状態などのアプリ内データ
  • 見た目の設定やアプリ内の保存情報

2. 利用目的

取得した情報は、以下の目的で利用します。

  • 買い物リスト、チェックリスト機能の提供
  • サジェスト機能や入力補助の提供
  • 進捗管理、表示カスタマイズ機能の提供
  • JSON 書き出し / 読み込みなどの保存機能の提供
  • 不具合対応、機能改善の参考

3. 広告について

ShopSuggest では広告配信のために第三者サービスを利用する場合があります。 その際、広告配信事業者が識別子等の情報を利用することがあります。

Pro 機能を利用すると広告を非表示にできます。

4. アプリ内課金について

ShopSuggest では、Pro 機能の解放や任意の応援購入のために Apple のアプリ内課金を利用します。 課金情報の管理は Apple により行われ、開発者がクレジットカード情報等を直接取得することはありません。

5. JSON 書き出し / 読み込みについて

Pro 機能では JSON 書き出し / 読み込みを利用できます。 ユーザーが書き出したデータの管理、保存、共有はユーザー自身の責任で行ってください。

  • 書き出した JSON ファイルには、ユーザーが入力した内容が含まれる場合があります
  • 共有や送信を行う際は内容を確認のうえご利用ください

6. 第三者提供

開発者は、法令に基づく場合を除き、ユーザーの情報を第三者へ提供しません。 ただし、広告配信、課金処理などにおいて第三者サービスが利用される場合があります。

7. 安全管理

開発者は、取り扱う情報の漏えい、滅失、毀損の防止に努めます。 ただし、ユーザー自身が書き出したファイルや外部保存先については、ユーザー自身で管理をお願いします。

8. 改定について

本ポリシーは、必要に応じて内容を変更することがあります。 変更後の内容は、このページに掲載した時点から適用されます。

9. お問い合わせ

本ポリシーに関するお問い合わせは、サポートページ記載の連絡先までお願いします。
