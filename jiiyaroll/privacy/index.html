じいやロール Privacy Policy

プライバシーポリシー

じいやロールでは、安心して遊べるように、データの取り扱い、Game Center、外部サービス、お問い合わせ先について案内しています。

基本方針

じいやロールは、アプリの提供、改善、サポートのために必要な範囲で情報を取り扱います。

1. 取得する情報について

じいやロールでは、開発者がユーザーのゲーム内入力内容やプレイ内容そのものを独自に収集することはありません。 アプリ内で発生する進行状況、所持金、試行回数、雇用状態、設定などのゲームデータは、アプリを遊ぶためのデータとして端末内に保存される場合があります。

  • ゲームの進行状況
  • 所持金、試行回数、雇用状態、強化状態などのプレイデータ
  • アプリ内の設定や表示状態
  • Game Centerのリーダーボードや実績に送信されるスコア・達成情報

※ 開発者が、アプリ内のプレイ内容そのものを独自のサーバーへ送信して収集することはありません。

2. Game Centerについて

じいやロールでは、リーダーボードや実績のためにAppleのGame Centerを使用する場合があります。 Game Centerを利用する場合、スコア、ランキング、実績の達成状況などがAppleの仕組みを通じて処理されます。

Game Centerに関する情報の取り扱いについては、Appleが定めるプライバシーポリシーやGame Centerの設定をご確認ください。

3. 広告について

現在の公開準備段階では、じいやロールはアプリ内広告を前提としない構成で準備しています。 今後広告を導入する場合は、App Storeの表示や本ポリシーを必要に応じて更新します。

4. アプリ内課金について

現在の公開準備段階では、じいやロールはアプリ内課金を前提としない構成で準備しています。 今後アプリ内課金を導入する場合、課金処理および購入情報の管理はAppleの仕組みを通じて行われます。

5. 解析・計測について

現在の公開準備段階では、開発者が独自にプレイ内容を収集する解析・計測を前提としない構成で準備しています。 今後、品質向上や不具合把握のために解析または計測ツールを利用する場合は、必要に応じて本ポリシーを更新します。

6. 外部サービスについて

じいやロールでは、アプリ提供、Game Center、App Store、サポート対応などに関連して外部サービスを利用する場合があります。 外部サービスにおける情報の取り扱いについては、それぞれの提供元が定めるポリシーをご確認ください。

7. お問い合わせ

本ポリシーに関するお問い合わせは、サポートページ記載の連絡先までご連絡ください。

8. 改定について

本ポリシーは、必要に応じて内容を見直し、改定する場合があります。 重要な変更がある場合は、アプリまたは関連ページ上でお知らせします。
