よくある質問

はじめて遊ぶ時や、Game Center・実績・ランキングについて気になった時に確認しやすいよう、基本的な案内をまとめています。

どんなゲームですか？

じいやロールは、サイコロを振ってお金を稼ぎ、じいやを雇い、環境を改善しながら24時間の試行回数を積み上げるカジュアルシミュレーションゲームです。

24時間のタイマーはいつ始まりますか？

最初のロールを行ったタイミングで、24時間チャレンジが開始されます。開始後は、残り時間の中で試行回数や所持金を伸ばしていきます。

Game Centerは何に使いますか？

Game Centerは、リーダーボードと実績に使用します。総試行回数、最終所持金、日次・週次ランキングなどで結果を競えます。

実績が解除されません。

Game Centerにサインインしているか、通信状態が安定しているかを確認してください。条件達成直後に反映されない場合でも、少し時間を置くと反映されることがあります。

ランキングに反映されません。

Game Centerへの送信には通信状態やApple側の反映時間が影響する場合があります。しばらく待っても反映されない場合は、アプリの再起動や通信環境の確認をお試しください。

広告はありますか？

現在、アプリ内広告はありません。今後変更がある場合は、App Storeの表示やプライバシーポリシーで案内します。

ログインは必要ですか？

アプリ独自のアカウント登録は不要です。Game Centerのランキングや実績を使う場合は、AppleのGame Centerへのサインインが必要になる場合があります。

データの取り扱いはどこで確認できますか？

データの取り扱いについては、プライバシーポリシーをご確認ください。
